# Pengenalan

Selamat datang di Pedeve DMS! Panduan ini akan membantu Anda memulai menggunakan aplikasi.

## Apa itu Pedeve DMS?

Pedeve Document Management System adalah aplikasi web untuk mengelola dokumen dan data perusahaan dengan fitur:

- Manajemen perusahaan dengan hierarki multi-level
- Manajemen dokumen dengan sistem folder
- Laporan keuangan (RKAP & Realisasi)
- Manajemen pengguna dengan kontrol akses
- Sistem notifikasi

## Persyaratan

Untuk menggunakan aplikasi, Anda memerlukan:

- Akun pengguna yang sudah dibuat oleh administrator
- Browser modern (Chrome, Firefox, Safari, atau Edge)
- Koneksi internet

## Langkah Pertama

1. **Akses Aplikasi**
   - Buka aplikasi di URL yang diberikan oleh administrator
   - Pastikan Anda menggunakan browser yang kompatibel

2. **Login**
   - Masukkan username dan password Anda
   - Jika sudah setup 2FA, masukkan kode verifikasi

3. **Kenali Dashboard**
   - Setelah login, Anda akan melihat dashboard
   - Dashboard menampilkan informasi sesuai dengan peran Anda

## Peran Pengguna

Aplikasi memiliki beberapa peran dengan level akses berbeda:

| Peran | Deskripsi | Akses |
|-------|-----------|-------|
| **Administrator** | Administrator | Akses penuh ke semua fitur |
| **Admin** | Admin perusahaan | Akses terbatas pada perusahaan yang di-assign |
| **Manager** | Manager | Akses untuk melihat dan mengelola data perusahaan |
| **Staff** | Staff | Akses terbatas untuk input data |

## Navigasi

Menu navigasi utama terletak di bagian atas halaman:

- **Dashboard** - Tampilan utama dengan ringkasan data
- **Anak Perusahaan** - Manajemen data perusahaan
- **Dokumen** - Manajemen dokumen
- **Laporan** - Manajemen laporan keuangan
- **Pengguna** - Manajemen pengguna (hanya untuk admin)
- **Notifikasi** - Daftar notifikasi
- **Pengaturan** - Pengaturan akun dan aplikasi
