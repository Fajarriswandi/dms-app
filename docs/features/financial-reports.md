# Laporan Keuangan - Daftar Laporan

Halaman ini menampilkan semua laporan keuangan yang dapat Anda akses.

## Akses Halaman

- Klik menu **Laporan** di navigasi atas
- Atau akses langsung: `/reports`

## Tampilan

Halaman menampilkan tabel dengan kolom:

- **Periode**: Periode laporan (format: YYYY-MM)
- **Perusahaan**: Nama perusahaan
- **Revenue**: Total pendapatan
- **OPEX**: Total biaya operasional
- **NPAT**: Net Profit After Tax
- **Dividend**: Dividen
- **Financial Ratio**: Rasio keuangan
- **Status**: Status laporan (RKAP/Realisasi)
- **Aksi**: Tombol untuk melihat detail atau edit

## Fitur

### Pencarian

- Gunakan search box untuk mencari laporan
- Pencarian berdasarkan nama perusahaan atau periode

### Filter

- **Filter Periode**: Pilih periode tertentu
- **Filter Perusahaan**: Pilih perusahaan tertentu
- **Filter Status**: RKAP atau Realisasi

### Sorting

- Klik header kolom untuk sorting
- Sorting berdasarkan periode, revenue, dll

### Pagination

- Gunakan pagination untuk navigasi
- Pilih jumlah item per halaman

## Aksi

### Lihat Detail

1. Klik nama perusahaan atau periode di tabel
2. Atau klik tombol **Detail** di kolom Aksi
3. Anda akan diarahkan ke halaman detail laporan

### Edit Laporan

1. Klik tombol **Edit** di kolom Aksi
2. Anda akan diarahkan ke halaman edit
3. **Catatan**: Hanya dapat edit laporan perusahaan yang di-assign

### Tambah Laporan Baru

1. Klik tombol **Tambah Laporan** di atas tabel
2. Anda akan diarahkan ke halaman form tambah laporan
3. **Catatan**: Lihat panduan [Tambah Laporan](./add-report)

### Bulk Upload

1. Klik tombol **Bulk Upload** di atas tabel
2. Download template Excel
3. Isi template dengan data laporan
4. Upload file Excel
5. **Catatan**: Lihat panduan [Bulk Upload](./bulk-upload-reports)

## Status Laporan

Laporan memiliki dua status:

- **RKAP**: Rencana Kerja dan Anggaran Perusahaan (planning)
- **Realisasi**: Realisasi aktual (actual)

Sistem mendukung perbandingan RKAP vs Realisasi untuk analisis.

## Tips

- Gunakan filter untuk melihat laporan spesifik
- Gunakan bulk upload untuk input banyak laporan sekaligus
- Export data untuk analisis lebih lanjut
