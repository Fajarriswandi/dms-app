# Manajemen Dokumen - Detail Dokumen

Halaman detail menampilkan informasi lengkap tentang dokumen.

## Akses Halaman

- Klik nama dokumen di daftar dokumen
- Atau akses langsung: `/documents/{id}`

## Tampilan Detail

Halaman menampilkan:

### Informasi Dokumen

- **Judul**: Judul dokumen
- **Nomor Referensi**: Nomor referensi
- **Tipe Dokumen**: Tipe dokumen
- **Folder**: Folder tempat dokumen
- **Status**: Status dokumen (Aktif/Nonaktif)
- **File**: Nama file dan ukuran

### Metadata

- **Unit**: Unit atau departemen
- **Tanggal Terbit**: Tanggal dokumen diterbitkan
- **Tanggal Efektif**: Tanggal dokumen mulai efektif
- **Tanggal Kedaluwarsa**: Tanggal dokumen kedaluwarsa
- **Status Aktif**: Status aktif dokumen
- **Uploader**: User yang mengupload
- **Tanggal Upload**: Tanggal dokumen diupload
- **Tanggal Update**: Tanggal terakhir diupdate

### Preview Dokumen

- **Preview Area**: Menampilkan preview dokumen
- **Download**: Tombol untuk download dokumen

## Fitur

### Preview Dokumen

**Untuk Gambar (JPG, PNG):**
- Preview langsung ditampilkan di halaman
- Klik gambar untuk preview fullscreen
- Zoom in/out dengan mouse wheel

**Untuk PDF:**
- Preview PDF ditampilkan di halaman
- Klik preview untuk fullscreen
- Navigasi halaman dengan tombol atau scroll

**Untuk Dokumen Lain (DOCX, XLSX, dll):**
- Tidak ada preview visual
- Hanya tombol download yang tersedia

### Download Dokumen

1. Klik tombol **Download**
2. File akan terdownload ke komputer Anda
3. File akan menggunakan nama asli saat upload

### Edit Dokumen

1. Klik tombol **Edit** di pojok kanan atas
2. Anda akan diarahkan ke halaman edit
3. **Catatan**: Hanya dapat edit metadata, tidak dapat edit file

### Delete Dokumen

1. Klik tombol **Delete** di pojok kanan atas
2. Konfirmasi penghapusan
3. **Catatan**: Hanya administrator yang dapat delete

### Lengkapi Metadata

Jika metadata belum lengkap:

1. Klik tombol **Lengkapi Meta Data**
2. Anda akan diarahkan ke halaman edit
3. Isi metadata yang belum lengkap
4. Simpan perubahan

## Tips

- Gunakan preview untuk melihat dokumen sebelum download
- Lengkapi metadata untuk memudahkan pencarian
- Cek tanggal kedaluwarsa untuk tracking
- Download dokumen penting sebagai backup

## Troubleshooting

### Preview Tidak Tampil

- Pastikan file adalah gambar (JPG, PNG) atau PDF
- File DOCX/XLSX tidak bisa di-preview, hanya download
- Cek koneksi internet jika preview tidak muncul
- Coba refresh halaman

### Tidak Bisa Download

- Cek koneksi internet
- Coba browser lain
- Hubungi administrator jika masalah berlanjut

### Metadata Tidak Lengkap

- Klik "Lengkapi Meta Data" untuk melengkapi
- Isi semua field yang diperlukan
- Simpan perubahan
