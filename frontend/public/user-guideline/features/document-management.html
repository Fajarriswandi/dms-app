Manajemen Dokumen - Daftar Dokumen ​

Halaman ini menampilkan semua dokumen yang dapat Anda akses.

Akses Halaman ​

  • Klik menu Dokumen di navigasi atas
  • Atau akses langsung: /documents

Tampilan ​

Halaman terdiri dari beberapa bagian:

Sidebar Kiri ​

  • Search: Pencarian dokumen
  • Folders: Daftar folder untuk navigasi
  • Activity: Aktivitas terbaru
  • Storage: Informasi penggunaan storage

Area Utama ​

Bagian Folders ​

Menampilkan folder dalam format grid atau list:

  • Grid View: Tampilan kartu folder
  • List View: Tampilan tabel folder
  • Add Folder: Tombol untuk menambah folder baru

Fitur Folder:

  • Klik folder untuk melihat dokumen di dalamnya
  • Double-click untuk membuka folder
  • Klik menu (3 dots) untuk rename atau delete folder

Bagian Recently Files ​

Tabel menampilkan file terbaru:

  • Nama File: Nama dokumen
  • Tipe: Tipe file (PDF, DOCX, dll)
  • Folder: Folder tempat dokumen
  • Uploader: User yang mengupload
  • Tanggal: Tanggal upload/update
  • Status Metadata: Status kelengkapan metadata

Fitur ​

Pencarian ​

  • Gunakan search box di sidebar untuk mencari dokumen
  • Pencarian berdasarkan nama file
  • Hasil pencarian ditampilkan di modal

Filter ​

  • Filter Folder: Pilih folder untuk melihat dokumen di folder tersebut
  • Filter Tipe: Filter berdasarkan tipe file
  • Filter Waktu: All Time, Last Month, Last Week

View Mode ​

  • Grid View: Tampilan kartu (default)
  • List View: Tampilan tabel

Sorting ​

  • Sort berdasarkan nama, tipe, tanggal update
  • Default: Sort berdasarkan tanggal update (terbaru dulu)

Pagination ​

  • Gunakan pagination untuk navigasi
  • Pilih jumlah item per halaman

Aksi ​

Lihat Detail Dokumen ​

  1. Klik nama dokumen di tabel
  2. Atau klik tombol "Meta Data ✓" / "Lengkapi Meta Data"
  3. Anda akan diarahkan ke halaman detail dokumen

Upload Dokumen ​

  1. Klik tombol Upload File di sidebar atau menu
  2. Anda akan diarahkan ke halaman upload
  3. Catatan: Lihat panduan Upload Dokumen

Tambah Folder ​

  1. Klik tombol Add Folder atau **Add new folder"
  2. Masukkan nama folder
  3. Pilih parent folder (opsional)
  4. Klik Buat untuk menyimpan

Rename Folder ​

  1. Klik menu (3 dots) di folder
  2. Pilih Rename
  3. Masukkan nama baru
  4. Klik Simpan

Delete Folder ​

  1. Klik menu (3 dots) di folder
  2. Pilih Delete
  3. Konfirmasi penghapusan
  4. Catatan: Hanya administrator yang dapat delete

Storage Information ​

Sidebar kanan menampilkan:

  • Total Storage: Total storage yang digunakan
  • Storage Breakdown: Breakdown berdasarkan tipe file
  • Progress Bar: Visualisasi penggunaan storage

Tips ​

  • Gunakan folder untuk mengorganisir dokumen
  • Gunakan search untuk menemukan dokumen dengan cepat
  • Lengkapi metadata dokumen untuk memudahkan pencarian
  • Gunakan filter untuk melihat dokumen spesifik

Langkah Selanjutnya ​

  • Detail Dokumen - Pelajari detail dokumen
