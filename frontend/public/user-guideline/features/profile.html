Profil Pengguna ​

Halaman ini menampilkan dan mengelola informasi profil Anda.

Akses Halaman ​

  • Klik nama pengguna di pojok kanan atas
  • Pilih Profil dari dropdown menu
  • Atau akses langsung: /profile

Tampilan ​

Halaman menampilkan informasi profil:

Informasi Dasar ​

  • Username: Username untuk login
  • Email: Email pengguna
  • Nama Lengkap: Nama lengkap (jika ada)
  • Role: Peran pengguna
  • Perusahaan: Perusahaan yang di-assign
  • Status: Status aktif/nonaktif

Informasi Tambahan ​

  • Telepon: Nomor telepon (jika ada)
  • Tanggal Bergabung: Tanggal akun dibuat
  • Terakhir Login: Waktu terakhir login

Edit Profil ​

Mengubah Informasi ​

  1. Klik Tombol "Edit"

    • Tombol berada di pojok kanan atas
    • Anda akan diarahkan ke form edit

  2. Edit Field yang Diizinkan

    • Nama Lengkap: Dapat diubah
    • Telepon: Dapat diubah
    • Email: Dapat diubah (jika diizinkan)
    • Password: Dapat diubah

  3. Simpan Perubahan

    • Klik "Update" untuk menyimpan
    • Perubahan akan tersimpan

Mengubah Password ​

  1. Di Form Edit Profil

    • Masukkan password lama
    • Masukkan password baru
    • Konfirmasi password baru

  2. Validasi

    • Password baru harus minimal 8 karakter
    • Password baru harus berbeda dengan password lama
    • Konfirmasi password harus sama

  3. Simpan

    • Klik "Update Password"
    • Password akan diubah
    • Anda perlu login ulang dengan password baru

Tips ​

  • Update informasi profil secara berkala
  • Gunakan password yang kuat dan unik
  • Jangan bagikan informasi login Anda

Troubleshooting ​

Tidak Bisa Edit Email ​

  • Email mungkin tidak bisa diubah oleh user sendiri
  • Hubungi administrator untuk mengubah email

Password Tidak Bisa Diubah ​

  • Pastikan password lama benar
  • Pastikan password baru memenuhi kriteria
  • Pastikan konfirmasi password sama dengan password baru

Langkah Selanjutnya ​
